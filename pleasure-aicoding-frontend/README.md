# PleaSureAICoding
## introduce
一款利用Vue3+TS+AntDesign+SpringBoot+MyBatis+Mysql+LangChain4j的前后端项目

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
